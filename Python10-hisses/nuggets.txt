Developed in <b>1990</b>, Python is one of the most popular <i>general-purpose</i> programming languages in modern times.


The term “general-purpose” simply means that Python can be used for a variety of applications and does not focus on any one aspect of programming. 
</p> <br>

To display data/information

print (data)


Use case:
	To display <strong>Strings</strong>(group of characters enclosed in single or double quotes) <br/>
print('Hello World') // Hello World
print("Hello World") // Hello World

	To display <strong>Numbers</>
print(50) // 50
print(1000) // 1000
print(3.142) // 3.142

	To display multiple data
print(50, 1000, 3.142, "Hello World", 'hello') // 50, 1000, 3.142, Hello World, hello 

By default Python prints <code>print</code> statements on new lines hence to join these  <code>print</code> statements use the  <code>end</code> attribute with whatever value you wish.
Use case:
print("Hello, come",  end=" ")
print("World") // Hello, come World

print("Hello", end=" ")
print("World") // Hello World

print("Hello", end="") 
print("World") // HelloWorld
